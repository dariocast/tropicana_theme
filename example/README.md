# Tropicana Theme Example

A Flutter application demonstrating the capabilities of the [Tropicana Theme](https://pub.dev/packages/tropicana_theme) package.

## Overview ðŸ“±

This example app showcases:

*   **Light & Dark Mode Switching:** Seamlessly toggle between the two theme modes to see how colors and typography adapt.
*   **Component Showcase:** Visualizes standard Material widgets styled with the Tropicana theme:
    *   **Buttons:** Elevated, Filled, Outlined.
    *   **Inputs:** TextFields with custom borders and focus states.
    *   **Surfaces:** Cards and Containers with themed backgrounds and shadows.
    *   **Typography:** Headlines and body text using Fraunces and Mulish fonts.
